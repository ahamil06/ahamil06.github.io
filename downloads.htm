Downloads

   Enumerated below are pieces of software which have been archived here due to the relative difficulty that one would find in retrieving them on the open internet. Most of these programs were found using the wayback machine. While not all of these may be hard to find, I have decided to host them for easy and safe access. Be sure to read any pertinent information that accompanies each download link.

Thank you!

  

 

Lego Digital Designer 4.3.12

3Diva Printscreen

 
   This is the last version of Lego Digital Designer which was released on November 18th, 2019. I likely retrieved this using the Wayback Machine. The software has been discontinued, though it still should function properly. For more information regarding its end of life, see the official LEGO website.
    I used this software to download 3D models from Lego Digital Designer. The program is intended to download any 3D images displayed on the screen as .3dxml files. This is yet another program that was likely downloaded from the Wayback Machine, but I have had difficulties in finding the archived page with recent searches.  

Adobe Wallaby

Windows Live Essentials 2012

 
   Adobe Wallaby was an experimental program created by Adobe Labs to convert flash animations to HTML-5. Its features are limited, shown in the official documentation here. Having its stable release in 2012, this program is well over a decade old now. Included in the .zip file are brief instructions to install a period-correct version of Adobe AIR and the Wallaby program itself.
    Windows Live Essentials is a discontinued suite of Microsoft programs for Windows systems. I used this download solely to accesss Windows Movie Maker. I cannot express enough gratitude for those running MajorGeeks.com who provided this download.   

Minecraft 1.6.2 Server

Minecraft 4K

 
   Released on July 8th, 2013, Minecraft 1.6.2 was the first version of the game I have played. This file allows one to host his own multiplayer server on Windows systems with a GUI.
    I found this on the Minecraft Omniarchive Discord server long ago. This was created by Notch for the 2010 Java 4k Contest. Note that Java is required to run this.
